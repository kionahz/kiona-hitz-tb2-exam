## Instructions to run my app
1. Clone my repository in a location of your choice
```
git clone https://github.com/kionahz/kiona-hitz-tb2-exam.git
```
2. Change the directory:
```
cd kiona-hitz-tb2-exam
```

3. Run the code
Mac:
```
python3 app.py
```
Windows:
```
python app.py
```
